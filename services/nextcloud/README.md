# Nextcloud Installation (LAMP Stack)

## Environment
- OS: Debian / Ubuntu Server
- Web Server: Apache
- Database: MariaDB / MySQL
- PHP 8.x
- Network: Local VM with port forwarding

## Installation Summary
1. Install Apache, MariaDB, PHP
2. Configure virtual host for Nextcloud
3. Create database and user for Nextcloud
4. Download and extract Nextcloud
5. Set file permissions
6. Complete web-based installation

## Notes
- Running on LAMP stack without Docker
- Used for learning Linux server administration
- Focus on manual configuration and troubleshooting
